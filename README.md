#### Take Away:
Using keyframes with animations! 
<code>
@keyframes lineOne-open {
    0% {
        transform: translateY(0px) rotate(0deg);
    }
    50% {
        transform: translateY(23px) rotate(0deg);
    }
    100% {
        transform: translateY(23px) rotate(45deg);
    }
}
</code>
